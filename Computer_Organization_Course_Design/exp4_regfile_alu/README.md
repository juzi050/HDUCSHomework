# 实验 4 寄存器堆与运算器设计

本工程面向 HCS-A02 教学 FPGA 开发板，实现 32x32 位寄存器堆、8 功能 ALU，以及“寄存器读出 -> ALU 运算 -> 结果写回”的完整数据通路。

## 文件说明

- `src/regfile.v`：32 个 32 位寄存器，双异步读、单同步写，复位清零，`R0` 恒为 0。
- `src/alu.v`：支持加、减、与、或、异或、左移 1 位、右移 1 位、取反，并输出 `ZF/CF/OF/SF`。
- `src/alu_reg.v`：集成寄存器堆和 ALU，支持外部数据装载和 ALU 结果写回。
- `src/top.v`：HCS-A02 顶层，连接开关、按键、LED 和 8 位数码管。
- `sim/*_tb.v`：寄存器堆、ALU、集成模块和顶层的自检仿真。
- `constrs/HCS_A02.xdc`：HCS-A02 完整引脚约束。
- `scripts/create_project.tcl`：创建 Vivado 工程。
- `scripts/run_all.tcl`：运行仿真、综合、实现并生成 bitstream。

## ALU 操作码

| `sw[35:32]` | 功能 | 结果 |
| --- | --- | --- |
| `0000` | 加法 | `A + B` |
| `0001` | 减法 | `A - B` |
| `0010` | 按位与 | `A & B` |
| `0011` | 按位或 | `A | B` |
| `0100` | 按位异或 | `A ^ B` |
| `0101` | 逻辑左移 | `A << 1` |
| `0110` | 逻辑右移 | `A >> 1` |
| `0111` | 取反 | `~A` |

`ZF` 表示结果为 0，`CF` 在加法中表示进位、减法中表示借位，`OF` 仅对有符号加减有效，`SF` 为结果最高位。

## 板卡操作

- `sw[31:0]`：外部写入数据。
- `sw[4:0]`：按键锁存地址时的地址输入。
- `sw[35:32]`：ALU 操作码。
- `bt[0]`：锁存 A 读地址。
- `bt[1]`：锁存 B 读地址。
- `bt[2]`：锁存写地址。
- `bt[3]`：把 `sw[31:0]` 写入当前写地址。
- `bt[4]`：把 ALU 结果写回当前写地址。
- `bt[5]`：循环切换 LED/数码管显示内容：`result -> A -> B`。
- `bt[7]`：复位，清空寄存器堆和顶层锁存状态。

显示规则：

- `ld[31:0]`：当前显示值。
- `ld[32]`：`ZF`。
- `ld[33]`：`CF`。
- `ld[34]`：`OF`。
- `ld[35]`：`SF`。
- `an[7:0]`、`seg[6:0]`、`dp`：用 8 位数码管显示当前显示值的 8 位十六进制数。

## 示例操作

1. 设置 `sw[4:0]=1`，按 `bt[2]` 锁存写地址 R1。
2. 设置 `sw[31:0]=3`，按 `bt[3]` 写入 R1。
3. 设置 `sw[4:0]=2`，按 `bt[2]` 锁存写地址 R2。
4. 设置 `sw[31:0]=5`，按 `bt[3]` 写入 R2。
5. 设置 `sw[4:0]=1`，按 `bt[0]` 锁存 A 读地址。
6. 设置 `sw[4:0]=2`，按 `bt[1]` 锁存 B 读地址。
7. 设置 `sw[35:32]=0000`，LED 和数码管应显示 `00000008`。
8. 设置 `sw[4:0]=3`，按 `bt[2]` 锁存写地址 R3，再按 `bt[4]` 把结果写回 R3。

## 运行命令

在 PowerShell 中进入本目录后执行：

```powershell
& "E:\Vivado\2025.1\Vivado\bin\vivado.bat" -mode batch -source .\scripts\run_all.tcl
```

生成的 Vivado 工程位于 `vivado_project`，输出文件位于 `build`。下载到板子的 bit 文件为：

```text
build\RegisterFile_Experiment.bit
```
